其中“3306、192.168.3.6”替换为自己需要的 IP 和端口：

  • listenaddress：本机 IP
  • connectport：本机端口
  • connectaddress：转发 IP（其他电脑）
  • connectport：转发端口（其他电脑）

可用此方法，通过外网服务器在外网访问内网的数据库。比如云服务器有公网 IP，但云数据库一般只有内网 IP，此方法转发后，可实现外网能够访问云数据库
如果是服务器想转内网，listenaddress 为内网 IP

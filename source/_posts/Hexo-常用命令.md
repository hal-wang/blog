| 命令                 | 作用       | 简写 | 备注                |
| -------------------- | ---------- | ---- | ------------------- |
| hexo init            | 初始化     |
| hexo new "文章名"    | 新建文章   | n    |
| hexo new page "页名" | 新建页面   |      | 页面中包含文章      |
| hexo generate        | 生成       | g    | 类似于编译          |
| hexo server          | 预览       | s    | 地址 localhost:4000 |
| hexo deploy          | 部署       | d    | 发布到 github       |
| hexo clean           | 清除缓存   |      | 删除文章后需用      |
| hexo g -d            | 生成并发布 |      | 组合命令            |

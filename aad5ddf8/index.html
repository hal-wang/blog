本文主要针对 Vue2，有些内容适合 Vue3，但有些不适合。Vue3 对 TS 的支持已经很友好了，而且也有相应的文档。但在 Vue2 中，使用 ts 需要一定写法才能更优雅。

创建项目

使用 Vue CLI 4 创建，可以用 vue create proj-name 或者 vue ui，按提示，选择 TS 模板。

使用 Vuex
